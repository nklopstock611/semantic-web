Execute server:

cd firstTask/pdf-analyzer

- First, clone and install the GROBID Python Client:

git clone https://github.com/kermitt2/grobid_client_python
cd grobid_client_python
python3 setup.py install

- Open ports and execute server:

docker-compose up -d
python grobid_server.py

- A chromedriver.exe is needed to run webscraper.py
